## üöÄ Installation
Follow these steps to get a local copy up and running:

1.  **Clone the Repository**
    ```bash
    git clone [https://github.com/ZakariaL-dev/Calculator.git](https://github.com/ZakariaL-dev/Calculator.git)
    ```
2.  **Navigate to the Project Directory**
    ```bash
    cd Calculator
    ```
3.  **Launch the App**
    Simply open the `index.html` file in your preferred browser:
    ```bash
    open index.html
    ```

## üìñ Usage
1.  Launch the application in your browser.
2.  Use the on-screen buttons or your keyboard to enter numbers.
3.  Press `=` or `Enter` to see the result instantly.
4.  Use the `C` or `AC` button to clear the display for a new calculation.
